# MERN-Project: MomoCuisine

**Live Demo:** [https://momocuisine.vercel.app](https://momocuisine.vercel.app)

MomoCuisine is a **MERN stack web application** for ordering momos and other food items. Users can browse the menu, add items to the cart, and place orders. Only **authenticated users** can complete orders via **eSewa** payment integration.
